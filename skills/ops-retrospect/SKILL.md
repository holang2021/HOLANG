# OPS 质量复盘器

## 触发词
"复盘"、"质量回顾"、"总结版本"、"版本总结"

## 流程
1. 输入版本号 → 自动汇总Bug数据
2. 分析: 按严重度/模块/根因分布
3. 计算效率指标: MTTR / 重开率 / 逃逸率
4. 提炼: 流程问题 / 需求问题 / 研发问题 / 测试覆盖问题
5. 输出改进建议 + 规范沉淀 (ADR)
